# Interactive-McDonald-s-Sales-Dashboard-
 McDonald’s Sales Dashboard 🍟📊
Excited to share a project I recently completed—a dynamic and interactive Excel dashboard built using real-world McDonald's sales data!

🔍 This dashboard dives deep into:

💰 Total sales revenue by item category

📆 Daily order volumes

🏆 Top-performing menu items

📈 Monthly revenue trends

⏰ Order patterns by time of day

📋 Average items per order

🔝 Top 5 menu item comparisons

✅ Before diving into analysis, I performed data cleaning (handled duplicates & missing values using forward/backward fill techniques) to ensure reliable insights.

📌 Bonus Insights: I added a few custom metrics and visualizations to uncover customer behavior patterns and seasonal preferences.

📄 Also included an executive summary highlighting key findings and strategic recommendations for stakeholders.

This project was a great blend of:

Power Query for data transformation

PivotTables & slicers for interactivity

Custom visualizations for clarity
